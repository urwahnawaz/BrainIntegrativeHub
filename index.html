Welcome to NeuroTranscript

Developed by Voineagu Lab, last updated October 2021.

NeuroTranscript is an integrative web-based resource of circRNA expression in the human brain, which allows interactive exploration of circRNA expression across multiple datatsets, including circRNA expression in bulk brain tissue, circQTLs, circRNA expression during neuronal differentiation and cellular maturation. NeuroTranscript also allows the upload of user-provided circRNA expression data and its exploration in the integrative platform.

To start exploring the data in NeuroTranscript, use the Gene Set search or Single Gene search tabs.

To view the source code/license and documentation visit the GitHub repository.
